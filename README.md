🧠 Sobre o projeto
O portfólio foi construído com HTML5, CSS3 e JavaScript puro, sem frameworks, para demonstrar domínio das tecnologias base da web. Ele reflete minha identidade visual e meu objetivo de me tornar um desenvolvedor completo.

Funcionalidades principais
Seletor de temas: 10 temas diferentes (Clássico, Escuro, Rainbow Road, Hacker News, etc.) com persistência via localStorage.

Navegação em SPA (Single Page Application): troca de seções (início, projetos, sobre) sem recarregar a página, usando history.pushState.

Página de projetos:

Listagem de cards com os principais projetos.

Visualização detalhada de cada projeto com galeria de imagens e descrições.

Lightbox para ampliar imagens.

Seção "Sobre": texto atualizado com minha formação, jornada, objetivos e interesses pessoais.

Design responsivo: adaptado para mobile, tablet e desktop.

Animações suaves e transições.

🛠️ Tecnologias utilizadas
HTML5

CSS3 (variáveis, flexbox, grid, animações)

JavaScript (ES6+)

Font Awesome para ícones

Google Fonts (Playfair Display, Lobster)
